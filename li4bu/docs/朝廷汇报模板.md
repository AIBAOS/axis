# 朝廷汇报模板

**汇报周期：** 每日卯时（08:00）/ 酉时（18:00）
**汇报频道：** #明镜高悬
**统筹：** 吏部尚书

---

## 📋 标准汇报格式

### 单任务场景

```markdown
**【部门名称】** 

**状态：** 🟢 正常 / 🟡 风险 / 🔴 阻塞 / ⏸️ 待命

**当前任务：**
- [任务名称] - 进度 XX% - 预计 [日期] 完成

**阻塞节点：**
- [如有] 具体阻塞问题 + 需要哪个部门协助

**交付期限：**
- [任务名称] → [YYYY-MM-DD]
```

### 多任务场景（一人/一部多职）

```markdown
**【部门名称】** 

**整体状态：** 🟢 正常 / 🟡 风险 / 🔴 阻塞 / ⏸️ 待命
**负载情况：** 正常 / 偏高 / 过载

**任务列表：**（按优先级排序）

| 优先级 | 任务名称 | 进度 | 预计完成 | 状态 |
|--------|----------|------|----------|------|
| P0 | [任务 A] | 80% | 2026-03-15 | 🟢 |
| P1 | [任务 B] | 30% | 2026-03-20 | 🟢 |
| P2 | [任务 C] | 0% | 2026-03-25 | ⏸️ |

**阻塞节点：**
- [任务 A] 需要 @工部 提供服务器（预计 3/14 前）

**时间分配：**
- 任务 A：60%
- 任务 B：30%
- 任务 C：10%

**交付期限：**
- 🚨 P0 [任务 A] → 2026-03-15（关键路径）
- P1 [任务 B] → 2026-03-20
- P2 [任务 C] → 2026-03-25
```

## 📊 状态说明

- 🟢 **正常** - 任务按计划推进
- 🟡 **风险** - 可能延期，需关注
- 🔴 **阻塞** - 无法推进，需立即协调
- ⏸️ **待命** - 无任务，等待分配

## 🔔 跨部协作格式

当需要其他部门协助时，按此格式呼叫：

```
@【部门】协作请求

【当前进度】简述当前进展
【阻塞节点】具体卡点 + 需要对方做什么
【交付期限】期望完成时间
```

**示例：**
```
@工部 协作请求

【当前进度】兵部正在开发新模块，需部署环境
【阻塞节点】缺少测试服务器，无法部署验证
【交付期限】需在本周五前就绪
```

## 📝 使用说明

1. **各部门**每日按时在 #明镜高悬 汇报
2. **吏部**汇总更新进度看板
3. **都察院**可抽查汇报真实性
4. **内阁**审议重大延期/阻塞事项
5. **司礼监**传达主子谕令

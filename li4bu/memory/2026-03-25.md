### API 端点
1. `PUT /api/v1/files/rename` — 重命名文件/文件夹
   - 请求体：`{ "path": "xxx", "newName": "xxx" }`
   - 同名冲突 → 409 Conflict
   - 源路径不存在 → 404 Not Found
   - 成功 → 200 OK + 新路径信息

2. `PUT /api/v1/files/move` — 移动文件/文件夹
   - 请求体：`{ "path": "xxx", "newPath": "xxx" }`
   - 目标已存在 → 409 Conflict
   - 源路径不存在 → 404 Not Found
   - 成功 → 200 OK + 新路径信息

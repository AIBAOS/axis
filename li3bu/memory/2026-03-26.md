### 技术要点

#### 认证模式
- **登录用户可访问**: 列表/详情接口（任意 JWT Token）
- **仅 admin 可访问**: 创建/删除接口（需要 admin 角色）

#### 错误处理规范
- 401: UNAUTHORIZED - 未认证或 Token 无效
- 403: FORBIDDEN - 权限不足（非 admin 角色）
- 404: NOT_FOUND - 资源不存在（VOLUME_NOT_FOUND/SNAPSHOT_NOT_FOUND 等）
- 409: CONFLICT - 名称已存在（VOLUME_EXISTS/SNAPSHOT_EXISTS）
- 400: BAD_REQUEST - 参数无效/容量不足/快照受保护

#### 返回字段规范
- 成功响应：`{ success: true, data: {...}, pagination: {...} }`
- 错误响应：`{ success: false, error: "...", code: "..." }`

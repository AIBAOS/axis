### 场景 1: 边界测试

#### 1.1 分页参数边界测试

**测试内容：** page=0, page=-1, per_page=0, limit=0

**发现问题：**

| Bug ID | 描述 | 严重度 | 状态 |
|--------|------|:------:|:----:|
| **#72** | 整数下溢：page=0 时 (page-1)*page_size 导致整数下溢 | 🔴 高 | ✅ 已修复 |

**Bug #72 详情：**

```rust
// 问题代码
let page = query.page.unwrap_or(1);  // 用户传入 page=0 时仍可能为 0
let offset = (page - 1) * page_size; // page=0 时 (0-1) 导致整数下溢
```

**修复方案：**
```rust
// 修复后
let page = query.page.unwrap_or(1).max(1);  // 确保 page >= 1
let page_size = query.page_size.unwrap_or(20).max(1);  // 确保 page_size >= 1
```

**影响文件：** 24 个 handlers 文件

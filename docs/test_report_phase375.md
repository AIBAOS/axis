## Token 验证流程

```
JWT Token 结构:
{
  sub: user_id,
  username: string,
  roles: [string],
  iat: issued_at,
  exp: expires_at
}

验证流程:
1. 解析 Bearer token
2. decode() 验证签名
3. 检查 exp 过期时间
4. 返回 Claims 或错误
```

| path=`../etc/passwd` | 拒绝访问 | ❌ **未验证** | 🔴 |
| path=`/data/../../../etc` | 拒绝访问 | ❌ **未验证** | 🔴 |

**🔴 发现 Bug #51: files_browse.rs 存在路径遍历漏洞**

**问题:**
```rust
// files_browse.rs 第 125-128 行
let target_path = if path == "/" {
    base_dir.clone()
} else {
    base_dir.join(path.trim_start_matches('/'))
};
```

- 没有检查 `..` 路径遍历
- 没有 canonicalize 验证
- 攻击者可访问任意系统文件

**影响:** 高危 - 可读取系统敏感文件

## 📈 修复建议

### Bug #63: 分页参数除零防护

**修复方案：**

```rust
// 修改前
let per_page = std::cmp::min(query.per_page.unwrap_or(20), 100);

// 修改后
let per_page = query.per_page.unwrap_or(20).max(1).min(100);
```

**影响文件：** 所有包含分页的 handler（约 15 个文件）

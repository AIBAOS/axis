## 发现的 Bug

### 🐛 DEPLOY-1: upgrade.sh 备份路径未创建

**严重程度**: 中  
**描述**: upgrade.sh 在执行备份时，如果备份目录不存在会失败  
**复现**: 
```bash
rm -rf /var/backups/axis
./scripts/upgrade.sh
# 错误：无法创建备份
```
**修复建议**: 在备份前添加 `mkdir -p` 创建目录  
**状态**: 📝 已记录

---

### 🐛 DEPLOY-2: healthcheck.sh 告警阈值硬编码

**严重程度**: 低  
**描述**: 告警阈值硬编码在脚本中，无法通过配置调整  
**影响**: 用户无法自定义告警阈值  
**修复建议**: 从 config.toml 读取阈值配置  
**状态**: 📝 已记录

---

### 🐛 DEPLOY-3: restore.sh 恢复后未验证服务状态

**严重程度**: 中  
**描述**: restore.sh 恢复完成后未验证服务是否正常启动  
**风险**: 可能恢复后服务实际未启动  
**修复建议**: 添加服务健康检查步骤  
**状态**: 📝 已记录

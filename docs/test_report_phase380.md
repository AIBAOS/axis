## 📝 详细分析

### 前端错误处理

WebUI 所有 API 调用均有 try-catch 包裹：

```typescript
// api/index.ts
apiClient.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) {
      localStorage.removeItem('jwt_token')
      window.location.href = '/login'
    }
    return Promise.reject(error)
  }
)
```

## 🔴 Bug #78: 错误信息泄露数据库错误

### 漏洞描述

**严重度：** 🟠 中

**影响：** 多个 handler 文件

### 漏洞详情

**问题：** 错误响应中包含原始数据库错误信息

**影响文件：**
- `apps.rs`: "查询应用列表失败: {}", e
- `backups_create.rs`: "Failed to create backup task: {}", e
- `containers_detail.rs`: "查询容器失败：{}", e
- `containers_list.rs`: "查询容器列表失败：{}", e

**攻击向量：**
```
// 攻击者可触发特定错误获取数据库结构信息
GET /api/v1/apps?filter='; SELECT * FROM users--
```

**影响：**
- 泄露数据库结构信息
- 泄露表名、字段名
- 辅助 SQL 注入攻击

### 修复方案

**修复前：**
```rust
Err(e) => HttpResponse::InternalServerError().json(json!({
    "message": format!("查询应用列表失败: {}", e)  // ❌ 泄露细节
}))
```

**修复后：**
```rust
Err(e) => {
    log::error!("查询应用列表失败: {}", e);  // ✅ 记录日志
    HttpResponse::InternalServerError().json(json!({
        "message": "Internal server error"  // ✅ 泛化错误
    }))
}
```
